Here is the verilog code for the frequency divider module:

```verilog
module freq_div (
    input CLK_in,
    input RST,
    output reg CLK_50,
    output reg CLK_10,
    output reg CLK_1
);

reg [3:0] cnt_10;
reg [6:0] cnt_100;

always @(posedge CLK_in or posedge RST) begin
    if (RST) begin
        CLK_50 <= 1'b0;
        CLK_10 <= 1'b0;
        CLK_1 <= 1'b0;
        cnt_10 <= 4'b0;
        cnt_100 <= 7'b0;
    end else begin
        // Generating CLK_50
        CLK_50 <= ~CLK_50;
        
        // Generating CLK_10
        if (cnt_10 == 4'b1000) begin
            CLK_10 <= ~CLK_10;
            cnt_10 <= 4'b0;
        end else begin
            cnt_10 <= cnt_10 + 1;
        end
        
        // Generating CLK_1
        if (cnt_100 == 7'b1100010) begin
            CLK_1 <= ~CLK_1;
            cnt_100 <= 7'b0;
        end else begin
            cnt_100 <= cnt_100 + 1;
        end
    end
end

endmodule
```

This code implements the frequency divider as described, with the input clock frequency of 100MHz and output frequencies of 50MHz, 10MHz, and 1MHz. The counters `cnt_10` and `cnt_100` are used to keep track of the division logic for generating the output frequencies.
